NAME
	/hangman - play a game of hangman
	/h - alias

COMMANDS
	/hangman | /h 				...Start a new public game; there can be only 1 public game per room.
	/hangman [letter] | [word]	...Guess a letter or the whole word
	/hangman [-p] | [-private]	...Start a private game; while playing a private game you cannot play a public game.
